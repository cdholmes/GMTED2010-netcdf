# Global elevation data in NetCDF format

This repository contains [Global Multi-resolution Terrain Elevation Data (GMTED2010)](https://www.usgs.gov/coastal-changes-and-impacts/gmted2010) in NetCDF4 format. 

[USGS distributes GMTED2010](https://topotools.cr.usgs.gov/gmted_viewer/gmted2010_global_grids.php) in ArcGrid format. The NetCDF4 files here were converted from the source using GDAL script in this repository. 

Seethe [GMTED2010 webpage](https://www.usgs.gov/coastal-changes-and-impacts/gmted2010) for a description of the dataset.
